# Hospital Management System (SQL)

## 📌 Project Overview
This project is a **Hospital Management System** relational database designed using **MySQL**. The main objective is to effectively manage and organize hospital operational data, specifically focusing on patients, doctors, and their appointments.

## 🚀 Key Features
* **Database Schema:** Created structured tables to handle data for Patients, Doctors, and Appointments.
* **Relational Integrity:** Implemented **Primary Keys** and **Foreign Keys** to establish and maintain strong relationships between entities.
* **Complex Queries:** Utilized `JOIN` statements to extract meaningful insights, such as linking patient records with their assigned doctors and specializations.

## 🛠️ Tech Stack
* **Database:** MySQL
* **Tool:** MySQL Workbench
